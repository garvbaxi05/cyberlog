Overview

This Flutter project displays a simple Home Screen dashboard using two styled cards.
It demonstrates basic UI layout, widget styling, and structuring using Flutterâ€™s core widgets.

Features Implemented
1. MaterialApp + HomeScreen Setup

The app starts with a MaterialApp wrapped inside the MyApp widget.
HomeScreen is set as the default landing page, and the debug banner is disabled.

2. AppBar

The screen includes a standard AppBar with the title "Home", following Material Design guidelines.

3. Dashboard Layout (Column + Containers)

The dashboard section is built using:

A Column to stack elements vertically

Padding and margin for spacing

Two styled cards, each made with a Container widget

These cards act as placeholders for future dashboard features.

UI Components Breakdown
Dashboard Card 1

Background color: Light Blue (Colors.blue[100])

Padding: 20

Border radius: 8

Displays: "Dashboard Card 1"

Dashboard Card 2

Background color: Light Green (Colors.green[100])

Same styling as card 1

Displays: "Dashboard Card 2"

Both cards demonstrate how to create a clean dashboard tile using Container + BoxDecoration.
