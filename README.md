is a simple Flutter application that provides a clean dashboard interface with multiple feature sections. The app focuses on cybersecurity awareness by presenting short, readable cyber tips in a distraction-free layout.

Features

Dashboard layout using a two-column grid

Soft colored cards with rounded corners

Smooth navigation between screens

Dedicated Cyber Tip screen

Minimal text-focused design with no emojis or decorative icons

App Overview
Dashboard Screen

The dashboard displays four main sections:

Daily Log

Cyber Tips

Device Security

Notes

Each section is represented by a large tappable card for quick navigation.

Cyber Tip Screen

The Cyber Tip screen displays short awareness phrases inside a centered card.
Users can tap the “New Tip” button to view another phrase.

The design is intentionally simple to keep the user focused on the content.
